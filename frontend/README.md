# Clarity – Behavioral Analysis Web Application

## 📌 Project Description
Clarity is a web-based application that analyzes user-submitted personal situations and classifies them into Healthy, Clarification, or Concerning patterns using a rule-based scoring system.

The system uses authentication, protected routes, and database storage to manage and analyze user data securely.

---

## 🛠 Tech Stack

### Frontend
- React.js
- React Router
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

---

## ⭐ Features
- User Registration & Login Authentication
- Logout Functionality
- Protected Routes
- Submit Situation for Analysis
- Keyword-Based Behavioral Scoring
- Automatic Classification System
- Store Analysis Results in Database
- Dashboard with Results Display

---

## Installation

Clone the Repository:
git clone <your-repository-link>
cd project-folder

Install Frontend Dependencies:
cd frontend
npm install

Install Backend Dependencies:
cd backend
npm install

---
## Running the Project

Start Backend:
cd backend
npm start

Start Frontend:
cd frontend
npm start
